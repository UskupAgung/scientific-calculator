# scientific-calculator
Kalkulator Saintifik 7-kolom responsif, dibangun menggunakan Vanilla JavaScript, HTML, dan CSS Grid. Proyek ini mereplikasi fungsionalitas kalkulator saintifik modern, menampilkan kemampuan dalam manipulasi DOM, pengelolaan state aplikasi yang kompleks, dan implementasi fungsionalitas matematika tingkat lanjut (Trigonometri, Logaritma, Eksponen).
